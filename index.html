$(expr, [context])

概述

选择器，通过querySelectorAll选出符合条件的元素并构造iQuery对象

参数

expr(String)

用来查找的字符串

context(Element)

作为待查找的 DOM 元素集、文档或 iQuery对象。

$(html)

概述

通过HTML字符串构建dom并生成iQuery对象

参数

html(String)

html字符串，例: <span><a href='#'>test</a></span>"

$(elements)

概述

通过传入Dom Element构造iQuery对象

参数

elements(Element)

单个Dom Element或者Element数组

$(callback)

概述

$(document).ready()的简写。DOM加载完成后执行callback

参数

callback(Function)

当DOM加载完成后要执行的函数

touch(fn1, [fn2], [fn3], [compat])

参数

fn1(Function)

touchstart事件发生时触发的函数

fn2(Function)

touchend事件发生时触发的函数

fn3(Function)

touchmove事件发生时触发的函数

compat(Boolean)

若为true，则对普通浏览器兼容，上面三个Function分别追加作用在mousedown, mouseup, mousemove上

$.setup([options])

参数

options:

key-value object, 可选，详细见下

选项

icon(String)

添加到IPHONE屏幕时的图标地址，默认为空

iconGloss(Boolean)

是否为图标添加高光，默认为true

startup(String)

添加到IPHONE屏幕后启动的图片地址(460*320)，默认为空

viewport(Boolean)

iphone浏览器显示设置，若为true，则设置页面宽度为iphone宽度，并不可缩放。默认为true

fullScreen(Boolean)

添加到IPHONE屏幕后是否全屏运行。默认为true

statusBar(String)

添加到IPHONE屏幕后是否隐藏地址栏。默认为true

$.hideBar()

概述

载入页面后隐藏地址栏

$.scrollTo(Ypos, [speed])

概述

竖直方向平滑滚动到某一位置

参数

Ypos(Number)

要滚动到的Y坐标

speed(String)

滚动速度，可选："slow", "normal", "fast"。默认为"normal"

$.orient()

概述

返回window.orientation，返回值90,-90为横屏

$.isIphone()

概述

判断当前浏览器是否为iphone safari

$.isStandalone()

概述

判断是否已经添加到屏幕，作为app打开

true: 已作为app打开

false: 在浏览器中打开
